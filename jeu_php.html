Réalisation d'un jeu en PHP

Pierre / Feuille / Ciseaux

Présentation

L'objectif de ce projet était de refaire le célèbre jeu "Pierre / Feuille / Ciseaux" en utilisant le language PHP. Le joueur doit pouvoir choisir une des trois options et l'ordinateur en choisira une aléatoirement.

Fonctions

Pour réaliser ce jeu, nous devons réaliser différentes étapes : une fonction qui permettra au joueur de choisir son coup, une fonctions qui permettra à l'ordinateur de jouer et une fonction qui comparera les coups de chaucun et déclarera le vainqueur.

Les bases

                    
<?php
include('vues/header.php');
include('vues/form.php');
include('vues/footer.php');

Commençons déjà par créer la base du projet : "index.php".

On utilise "include()" pour y intégrer d'autres pages. La page "vues/header.php" s'occupera d'afficher un titre tandis que la page "vues/footer.php" s'occupera de fermer le html de la page. La page "vues/form.php" sera créer plus tard.

Permettre au joueur de jouer

                    
<h2>Jouer</h2>
<form action="jouer.php">
<fieldset>
    <legend>Faites votre choix</legend>
    <input type="radio"  name="choix" value="0">
    <label for="0">
        Pierre
    </label>
    <input type="radio"  name="choix" value="1">
    <label for="1">
        Feuille
    </label>
    <input type="radio" name="choix" value="2">
    <label for="2">
        Ciseaux
    </label>
</fieldset>

<button type="submit">Envoyer</button>
</form>

Voici la page "vues/from.php". Ici, on génère un formulaire en HTML qui agira sur la page "jouer.php".

Le joueur a le choix entre 3 cercles, il ne peut en cocher qu'un : "Pierre", "Feuille" ou "Ciseaux". Le choix du joueur renverra une valeur qui sera comparée à celle choisit par la machine. 0 correspond à "Pierre", 1 à "Feuille" et 2 à "Ciseaux".

La réponse sera envoyé en utilisant la méthode GET, visible dans l'URL, car aucune donnée sensible ne va être envoyé. La clé "choix" lui sera attribuée.

Générer la réponse de l'ordinateur

                    
/**
 * Retourne le choix de l'ordinateur
 * 
 * 0 Pierre
 * 1 Ciseau
 * 2 Feuille
 *
 * 
 * @return int choix de l'ordinateur
 */
function ordi_choix(): int {
    return random_int(0, 2);
}

Dans ce code, on créer une fonction appelée "ordi_choix" qui va générer un chiffre aléatoire entre 0 et 2, qui sera comparé avec le choix du joueur pour désigner le gagnant.

Comparer les réponses

                    
/**
* Fonction retournant le gagnant d'une manche.
* Le gagnant est codé comme suit:
* 0: égalité
* 1: Joueur_1 a gagné
* 2: Joueur_2 a gagné
*
* @param int $joueur_1 Choix du joueur 1
* @param int $joueur_2 Choix du joueur 2
* @return int Gagnant (voir ci-dessus)
*/
function gagnant (int $joueur_1, int $joueur_2): int
{
    if ($joueur_1 == 0){
        // Joueur 1 a joué Pierre
        switch($joueur_2){
            case 0: return 0; // Pierre - Pierre
            case 1: return 2; // Pierre - Feuille
            case 2: return 1; // Pierre - Ciseaux
        }
    }
    if ($joueur_1 == 1){
        // Joueur 1 a joué Feuille
        switch($joueur_2){
            case 0: return 1; // Feuille - Pierre
            case 1: return 0; // Feuille - Feuille
            case 2: return 2; // Feuille - Ciseaux
        }
    }
    if ($joueur_1 == 2){
        // Joueur 1 a joué Ciseaux
        switch($joueur_2){
            case 0: return 2; // Ciseaux - Pierre
            case 1: return 1; // Ciseaux - Feuille
            case 2: return 0; // Ciseaux - Ciseaux
        }
    }
}

Dans ce code, on créer un fonction appelée "gagnant" qui va comparer le choix de chaque joueur pour désigner le gagnant.

On remarque que, contrairement à la fonction précédante, les parenthèses derrière le nom sont remplis. Ce sont des paramètres. Les valeurs qui seront rentrées dans les paranthèses lorsque la fonction sera appelée seront placé dans les variables ci-contre ($joueur_1 et $joueur_2), ce qui permet d'utiliser ces valeurs dans la fonction.

Commencons la construction

                    
/**
* Pierre - Feuille - Ciseaux
* Dans tout le script, on va coder les choix des joueurs:
* 0: Pierre
* 1: Feuille
* 2: Ciseaux
*/

// Les inclusions de bibliothèques de fonctions sont placées en début de fichier
include('fonctions.php');

// On récupère le choix de l'utilisateur
$choix_utilisateur = $_GET['choix'];
// La machine joue au hasard
$choix_machine = ordi_choix();
// On détermine le gagnant
$gagnant = gagnant($choix_utilisateur, $choix_machine);

// On génère le HTML
include('vues/reponse.php')

Voici le contenu de la page "jouer.php". C'est celle avec laquelle le formulaire créé précédemment intéragit. On utilise "include()" pour intégrer la page "fonctions.php" définis précédemment et nécessaire si on veut les utiliser, ou la page de "reponse.php" que l'on verra plus tard.

Comme préciser dans le programme, on récupère la réponse du joueur grâce à la variable superglobale $_GET. Une variable superglobale est une variable prédéfinie par PHP qui a une portée globale sur le projet. Ce qui signifie qu'elle peut être utilisé partout quand cela est nécéssaire. Elle se comporte comme un tableau, on précise donc qu'on recherche la valeur avec la clé "choix". La réponse du joueur est placée dans la variable "$choix_utilisateur".

Ensuite, on appelle la fonction "ordi_choix()" pour générer la réponse de l'ordinateur et on place cette dernière dans la variable "$choix_machine".

Enfin, on appelle la fonction "gagnant()" en lui donnant le score de chacun. Elle va comparer les résultats puis le verdict sera placé dans la variable "$gagnant".
