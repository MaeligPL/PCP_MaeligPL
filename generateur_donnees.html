Contexte

Cette AP se faisait par groupe de 2, un étudiant SLAM et un étudiant SISR.

L'objectif pour les étudiants SLAM était de développer un script en Python Orienté Objet pour permettre aux étudiants SISR de générer des profils aléatoires de particuliers ou d'organisations.

Fichiers fournis

Il nous était fournis plusieurs fichiers :

  • Un fichier "GenerateurDonneesPerso.py" contenant la classe 'GenerateurDonneesPerso' avec des méthodes permettant de récupérer des données depuis des dictionnaires.
  • Un fichier "Test_csv_odoo.py" contenant la classe 'Test_csv_odoo' avec des méthodes permettant de tester des fichiers CSV.
  • Différents dictionnaires en .txt contenant des noms, des prénoms, des adresses, etc...

Générateur de clients — particuliers

from GenerateurDonneesPerso import GenerateurDonneesPerso
from random import randint

def clients_particuliers(chemin):
    with open(chemin, "w") as reset:
        reset.write('')

    with open(chemin, "a") as fichier:
        fichier.write("Titre;Nom;Prénom;Email;Téléphone;Mobile;Code postal;Ville;Rue;Est une société \n")

        generateur = GenerateurDonneesPerso()
        nb_utilisateurs = int(input('Combien voulez-vous de clients particulier ? Nb = '))

        for i in range(0, nb_utilisateurs):
            x = randint(1,2)
            if x == 1:
                titre = "Madame"
                estFeminin = True
            else:
                titre = "Monsieur"
                estFeminin = False
            nom = generateur.gen_nom_famille()
            prenom = generateur.gen_prenom(estFeminin)
            tel_fixe = generateur.gen_tel_fixe(10)
            mobile = generateur.gen_tel_mobile(10)
            ville = generateur.gen_commune()
            rue = generateur.gen_adresse_rue()

            ligne = titre+";"+nom+";"+prenom+";"+str.lower(prenom)+"."+str.lower(nom)+"@gmail.com"+";"
            ligne = ligne+tel_fixe+";"+mobile+";"+ville["codePostal"]+";"+ville["nom"]+";"+rue+";0\n"
            fichier.write(ligne)

from GenerateurDonneesPerso import GenerateurDonneesPerso

def clients_societes(chemin):
    with open(chemin, "w") as reset:
        reset.write('')

    with open(chemin, "a") as fichier:
        fichier.write("Nom;Type de société;Rue;Ville;Code postal;Téléphone;Mobile;Email;Secteur d'activité;N° TVA;SIRET;Est une société \n")

        generateur = GenerateurDonneesPerso()
        nb_clients = int(input('Combien voulez-vous de clients sociétés ? Nb = '))

        for i in range(0, nb_clients):
            nom = generateur.gen_nom_societe()
            rSociale = generateur.gen_raison_sociale()
            rue = generateur.gen_adresse_rue()
            ville = generateur.gen_commune()
            tel_fixe = generateur.gen_tel_fixe(10)
            mobile = generateur.gen_tel_mobile(10)
            email = str.lower(nom+'@gmail.com')
            activite = generateur.gen_activite_secteur()
            siret = generateur.gen_siret()
            tva = generateur.gen_numtva()
